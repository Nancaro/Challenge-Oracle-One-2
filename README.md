# Conversor Java

Este es un programa Java que permite realizar conversiones de divisas y temperaturas. El programa presenta un menú de opciones al usuario, donde puede elegir entre convertir divisas, convertir temperaturas o salir del programa.

## Características

- Conversión de Divisas:
  - Pesos a Euros
  - Pesos a Dólares
  - Pesos a Libras
  - Pesos a Yen
  - Pesos a Won
  - Pesos a Reales

- Conversión de Temperatura:
  - Celsius a Fahrenheit
  - Fahrenheit a Celsius

## Requisitos

- Java Development Kit (JDK) 8 o superior.
- Compilador de Java (javac) y entorno de ejecución Java (java) configurados correctamente.

## Cómo utilizar

1. Clona este repositorio o descarga los archivos en tu equipo.
2. Abre una terminal o línea de comandos y navega hasta el directorio donde se encuentran los archivos.
3. Compila el código fuente ejecutando el siguiente comando:

   ```shell
   javac App.java
